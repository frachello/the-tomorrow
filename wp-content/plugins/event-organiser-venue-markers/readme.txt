== Description ==

Allows use to select and upload custom markers for venues on Google maps. Choose from nearly 200 bundled icons or upload your own.

To assign a marker to a venue, go the the venue edit page and click 'Select a marker'. An existing marker can then be selected from the grid 
or you can click upload to upload a new venue marker.

Recommended marker dimensions: 32px x 37px.

Requires **WordPress 3.5++** && **Event Organiser 2.2+**. 

== Installation ==

Installation is standard and straight forward. 

1. Go to your 'Plug-ins' admin page, and go to 'Add New', 'Upload'
1. Select event-organiser-venue-markers.zip and click 'install now'
1. Activty the plug-in
1. Proceed to 'Event Organsier' under your site's Settings admin tab to enter the license key
